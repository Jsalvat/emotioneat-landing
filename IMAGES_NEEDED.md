## Design Guidelines

All images should:
- Match the EmotionEat brand colors:
  - Brand: #1890FF
  - Calm: #0EA5E9
  - Mint: #22C55E
  - Lavender: #A855F7
- Use the Inter font family
- Have rounded corners (border-radius: 1rem to 1.5rem)
- Convey calm, hope, and support
- Be optimized for web (compressed, WebP format when possible)

## Image Optimization Tips

1. **Compress images** before uploading:
   - Use TinyPNG, ImageOptim, or similar
   - Target: < 200KB per image

2. **Use WebP format** when possible:
   - Better compression than JPEG
   - Widely supported

3. **Provide 2x versions** for retina displays:
   - Name them: `image-name@2x.jpg`

4. **Add alt text** in the components for accessibility
